# Rokid Discord Bridge

自动构建的 Rokid 眼镜与 Discord Bot 连接测试应用。

## 功能
- 通过蓝牙/WiFi 连接 Rokid Glasses
- 与 Discord Bot 通信
- 语音输入 → Discord → OpenClaw

## 下载
从 [Releases](../../releases) 页面下载最新 APK

## 自行构建
```bash
# 克隆仓库
git clone https://github.com/yourusername/rokid-discord-bridge.git
cd rokid-discord-bridge

# 使用 Android Studio 打开并构建
# 或推送到 GitHub 自动构建
```

## 配置
1. 安装 APK 到手机
2. 授予蓝牙和位置权限
3. 连接到 Rokid 眼镜
4. 配置 Discord Bot Token
5. 开始对话！
